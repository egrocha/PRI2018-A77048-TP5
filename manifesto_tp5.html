Resumo:

Este trabalho envolve a criação de um servidor Node.js capaz de redirecionar utilizadores para as diversas páginas do Arquivo de Marchas.

Parte do desafio deste trabalho foi o fato de as obras estarem dispostas apenas em formato .json, o que significa que foi preciso desenvolver um script capaz de extrair informações relevantes para que possa ser feito um ficheiro .json com as informações necessárias para o índice.

Para criar as páginas HTML utilizamos a ferramenta Pug, disponibilizada através da plataforma npm, que trata de converter os ficheiros .json em ficheiros .html. Utilizamos também o ficheiro .css fornecido pelo site w3schools.
